API reference
=============

.. py:module:: chromophile

.. py:data:: cmap

   Chromophile color maps as Matplotlib :class:`Colormap
   <matplotlib.colors.Colormap>` objects.  The color maps may be
   accessed as dictionary items or as attributes of :data:`cmap`.
   This object is :data:`None` if Matplotlib cannot be imported.

.. py:data:: palette

   Chromophile color maps in Bokeh format.  The color maps may be
   accessed as dictionary items or as attributes of
   :data:`palette`.
